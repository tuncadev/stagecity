== Installation ==
1. Download the plugin (.zip file).
2. In WordPress Dashboard go to Plugins > Add New.
3. Upload Plugin by pressing the upload button.
4. Activate the plugin.
5. In [Ultimate Membership Pro](https://ultimatemembershippro.com) plugin dashboard go to navigation menu and find EXTENSIONS tab. Upon clicking the menuâ€™s item, a box with Extended Shortcodes module will be available.

For more details, visit our [documentation](https://store.wpindeed.com/documentation/extended-shortcodes-2/) page.
